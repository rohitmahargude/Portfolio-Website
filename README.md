# 🌌 Stellar Odyssey - 3D Space Portfolio

Welcome to **Stellar Odyssey**, an interactive 3D portfolio game and website. Embark on a journey through the stars as you scroll through different game levels ("Space Zones") displaying my professional background, projects, skills, and contact info.

Developed with **React**, **Three.js** (via React Three Fiber & Drei), **GSAP** for animations, and **TailwindCSS v4**.

---

## 🚀 Features

*   **Interactive 3D Universe:** Powered by `@react-three/fiber` and `@react-three/drei`, featuring a dynamic planetary system and realistic space backgrounds.
*   **Sci-Fi HUD (Heads-Up Display):** A custom HUD dashboard tracking your progress, active space zones, and sound controls.
*   **Warp Sound Effects:** Atmospheric sound effects triggered during zone travel to enhance immersion.
*   **Smooth Scroll Snapping:** Sections snap seamlessly to screen dimensions, providing a crisp, game-like exploration flow.
*   **Post-processing Bloom:** Glow effects on cosmic bodies using `@react-three/postprocessing` for high-fidelity space visuals.
*   **Fully Responsive:** Optimized controls and visual layout adapting to various screen dimensions.

## 🎮 Navigation Zones (Space Levels)

As you scroll down, you travel through different cosmic sectors:
1.  **Sektor 0: Launch Pad (Hero)** - Landing area and introductory terminal.
2.  **Sektor 1: Biosphere (About)** - Personal bio and mission details.
3.  **Sektor 2: Space Station (Experience)** - Chronological professional experience logs.
4.  **Sektor 3: Nebula (Projects)** - Showcasing deployed operations (projects) and archives.
5.  **Sektor 4: Asteroid Belt (Skills)** - Core capabilities, languages, and technical frameworks.
6.  **Sektor 5: Communications Hub (Contact)** - Direct transmission gateway to send signals.
7.  **Sektor 6: Mission Control (Footer)** - Closing coordinates and social terminals.

## 💻 Local Setup & Installation

To run this space station locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/YOUR_USERNAME/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the Local Space Server (Dev):**
    ```bash
    npm run dev
    ```
    Open your browser and navigate to `http://localhost:5173`.

4.  **Build for Production Deployment:**
    ```bash
    npm run build
    ```
